# Guía 02 — Vectores y Funciones

## Instrucciones generales

Resolver los ejercicios en parejas o en grupos, según la modalidad habitual de trabajo de la comisión.

En esta guía todos los vectores tienen un tamaño fijo, definido con `#define`. Cuando un ejercicio menciona "agregar" un elemento, el vector ya fue declarado con lugares de más para eso: no hay que cambiar el tamaño del vector en ningún momento, solo usar una variable que indique cuántos de esos lugares están ocupados actualmente.

Por ejemplo, si un vector se declara así:

```c
#define CAPACIDAD 8
int vector[CAPACIDAD];
int cantidadActual = 5;
```

El vector siempre tiene 8 lugares físicos reservados, desde el inicio del programa hasta el final. `cantidadActual` es la que cambia: indica cuántos de esos 8 lugares tienen un valor válido en cada momento. Nunca hay que escribir en una posición igual o mayor a `CAPACIDAD`.

## Ejercicio 1 — Carga y promedio

Cargar un vector de 10 enteros con valores aleatorios entre 1 y 50. Calcular el promedio de todos los elementos y mostrarlo.

Prototipos:

```c
void cargarVector(int vector[], int cantidad, int minimo, int maximo);
float calcularPromedio(int vector[], int cantidad);
void mostrarVector(int vector[], int cantidad, char *titulo);
```

## Ejercicio 2 — Valor más alto

Cargar un vector de 12 enteros con valores aleatorios entre 1 y 100. Encontrar el valor más alto del vector y mostrar también en qué posición se encuentra.

Prototipos:

```c
void cargarVector(int vector[], int cantidad, int minimo, int maximo);
int posicionDelMayor(int vector[], int cantidad);
```

## Ejercicio 3 — Contar pares e impares

Cargar un vector de 15 enteros con valores aleatorios entre 1 y 30. Contar cuántos elementos son pares y cuántos son impares. Mostrar ambos resultados.

Prototipos:

```c
void cargarVector(int vector[], int cantidad, int minimo, int maximo);
int contarPares(int vector[], int cantidad);
int contarImpares(int vector[], int cantidad);
```

## Ejercicio 4 — Agregar un elemento al final

Declarar un vector con capacidad para 8 elementos. Cargar los primeros 6 con valores aleatorios entre 1 y 40, dejando 2 lugares libres. Mostrar el vector con sus 6 elementos.

Solicitar al usuario un valor nuevo y agregarlo al final del vector, en la primera posición libre. Mostrar el vector después de agregar el valor, ahora con 7 elementos.

Antes de agregar, verificar que todavía haya lugar disponible (es decir, que la cantidad actual sea menor que la capacidad).

Prototipos:

```c
void cargarVector(int vector[], int cantidad, int minimo, int maximo);
int agregarAlFinal(int vector[], int cantidadActual, int valorNuevo);
void mostrarVector(int vector[], int cantidad, char *titulo);
```

## Ejercicio 5 — Quitar el último elemento

Usando un vector cargado con 10 elementos (capacidad de 10), quitar el último elemento del vector. No es necesario mover ningún valor: alcanza con que la cantidad de elementos válidos disminuya en uno. Mostrar el vector antes y después, indicando en cada caso cuántos elementos tiene.

Prototipos:

```c
void cargarVector(int vector[], int cantidad, int minimo, int maximo);
int quitarUltimo(int cantidadActual);
void mostrarVector(int vector[], int cantidad, char *titulo);
```
