# Guía 03 — Vectores, Funciones y Ordenamiento

## Instrucciones generales

Resolver los ejercicios en parejas o en grupos, según la modalidad habitual de trabajo de la comisión.

Todos los vectores tienen tamaño fijo, definido con `#define`. Cuando un ejercicio pida construir un vector nuevo a partir de otro (filtrar, fusionar, etc.), el vector de salida ya tiene la capacidad necesaria reservada de antemano: la función solo tiene que indicar, con un valor de retorno o un parámetro por puntero, cuántos de esos lugares quedaron realmente ocupados.

A partir de esta guía ya podés usar algoritmos de ordenamiento (Bubble Sort) cuando el ejercicio lo pida.

## Ejercicio 1 — Vectores paralelos: mejor promedio

Cargar dos vectores paralelos de 8 elementos: uno de enteros (`legajos`) y otro de reales (`notas`), donde `notas[i]` corresponde a `legajos[i]`. Encontrar y mostrar el legajo y la nota del alumno con la calificación más alta.

Prototipos:

```c
void cargarLegajos(int legajos[], int cantidad, int minimo, int maximo);
void cargarNotas(float notas[], int cantidad, float minima, float maxima);
int posicionDeLaMejorNota(float notas[], int cantidad);
```

## Ejercicio 2 — Porcentaje de aprobados

Reutilizando un vector de 10 notas reales (entre 1 y 10), calcular cuántas notas son mayores o iguales a 6 y qué porcentaje representan sobre el total de alumnos.

Prototipos:

```c
void cargarNotas(float notas[], int cantidad, float minima, float maxima);
int contarAprobados(float notas[], int cantidad);
float calcularPorcentaje(int cantidadAprobados, int cantidadTotal);
```

## Ejercicio 3 — Búsqueda lineal: ¿existe el valor?

Cargar un vector de 20 legajos (enteros). Solicitar un legajo a buscar e informar si existe o no en el vector, usando una función que devuelva un valor lógico (`bool` de `stdbool.h`).

Prototipos:

```c
void cargarLegajos(int legajos[], int cantidad, int minimo, int maximo);
bool existeLegajo(int legajos[], int cantidad, int legajoBuscado);
```

## Ejercicio 4 — Búsqueda con retorno de posición

Sobre el mismo tipo de vector de legajos, diseñar una función que devuelva la posición donde se encuentra un legajo dado, o -1 si no existe. Informar la posición encontrada o un mensaje de "no encontrado".

Prototipos:

```c
void cargarLegajos(int legajos[], int cantidad, int minimo, int maximo);
int buscarPosicion(int legajos[], int cantidad, int legajoBuscado);
```

## Ejercicio 5 — Filtrado a un segundo vector

Cargar un vector de 15 enteros con valores aleatorios entre -20 y 20. Construir un segundo vector que contenga solamente los valores positivos del primero (mayores a 0), e indicar cuántos elementos válidos quedaron en ese segundo vector.

Prototipos:

```c
void cargarVector(int vector[], int cantidad, int minimo, int maximo);
int filtrarPositivos(int origen[], int cantidadOrigen, int positivos[]);
void mostrarVector(int vector[], int cantidad, char *titulo);
```

## Ejercicio 6 — Invertir un vector en el mismo lugar

Cargar un vector de 12 enteros. Invertir el orden de sus elementos dentro del mismo vector, sin usar un vector auxiliar (intercambiando posiciones extremas entre sí, avanzando hacia el centro).

Prototipos:

```c
void cargarVector(int vector[], int cantidad, int minimo, int maximo);
void invertirEnElMismoVector(int vector[], int cantidad);
void mostrarVector(int vector[], int cantidad, char *titulo);
```

## Ejercicio 7 — Ordenamiento con Bubble Sort

Cargar un vector de 10 alturas (valores reales, entre 1.40 y 2.10). Ordenar el vector de forma ascendente usando Bubble Sort. Mostrar el vector antes y después de ordenar.

Prototipos:

```c
void cargarAlturas(float alturas[], int cantidad, float minima, float maxima);
void ordenarBubbleSortAsc(float vector[], int cantidad);
void mostrarVector(float vector[], int cantidad, char *titulo);
```

## Ejercicio 8 — Unión de dos vectores

Cargar dos vectores de enteros: uno de 8 elementos y otro de 5 elementos. Construir un tercer vector de 13 elementos que contenga primero todos los valores del primero y después todos los del segundo, sin ordenar.

Prototipos:

```c
void cargarVector(int vector[], int cantidad, int minimo, int maximo);
void fusionarVectores(int vectorA[], int cantidadA, int vectorB[], int cantidadB, int resultado[]);
void mostrarVector(int vector[], int cantidad, char *titulo);
```

## Ejercicio 9 — Detección de duplicados

Cargar un vector de 10 códigos enteros. Diseñar una función que devuelva un valor lógico indicando si existe al menos un valor repetido en el vector (sin necesidad de ordenarlo previamente).

Prototipos:

```c
void cargarVector(int vector[], int cantidad, int minimo, int maximo);
bool tieneDuplicados(int vector[], int cantidad);
```

## Ejercicio 10 — Integrador: carga, estadísticas, orden y búsqueda

Diseñar un programa modular para un vector de 15 puntajes enteros (entre 0 y 100) que:

- Cargue el vector solicitando los valores al usuario.
- Calcule y muestre el máximo, el mínimo y el promedio (con dos decimales).
- Ordene el vector de forma ascendente con Bubble Sort.
- Permita al usuario buscar un puntaje puntual e informe si existe y en qué posición del vector ya ordenado.

Cada una de las cuatro tareas tiene que resolverse en una función distinta. `main()` solo declara el vector y coordina las llamadas en orden, sin contener lógica de cálculo.

Prototipos:

```c
void cargarPuntajes(int puntajes[], int cantidad);
int calcularMaximo(int puntajes[], int cantidad);
int calcularMinimo(int puntajes[], int cantidad);
float calcularPromedio(int puntajes[], int cantidad);
void ordenarBubbleSortAsc(int puntajes[], int cantidad);
int buscarPosicion(int puntajes[], int cantidad, int puntajeBuscado);
```
